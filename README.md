# Hassan Jameel - Portfolio

A premium, modern, high-performance frontend developer portfolio built with React 18, Vite, Tailwind CSS (v4), GSAP, and Lenis smooth scrolling.

## Installation Instructions

1. **Prerequisites**
   Ensure you have Node.js installed (v18+ recommended).

2. **Install Dependencies**
   Run the following command in the project root directory:

   ```bash
   npm install
   ```

3. **Start Development Server**
   To start the local Vite development server:

   ```bash
   npm run dev
   ```

   Open your browser at `http://localhost:5173`.

4. **Build for Production**
   Ready to deploy? Build the project with:
   ```bash
   npm run build
   ```

## Tech Stack Overview

- **React 18 & Vite**: Ultra-fast development and build times.
- **Tailwind CSS v4**: Utility-first styling with built-in dark mode and lightning-fast compilation.
- **GSAP**: Industry-standard high-performance animations (scroll triggers, staggering, fade-ins).
- **Lenis**: Buttery smooth scrolling experience.
- **Lucide React**: Clean, consistent vector icons.

## Modification Guide

- **Content**: Update texts in the respective `src/sections/*.jsx` files.
- **Styling**: Tweak core colors in `src/index.css` under the `@theme` block.
- **Images**: Replace Unsplash placeholders inside `src/sections/Projects.jsx` with your own hosted or imported assets.
- **Contact Details**: Update `src/sections/Contact.jsx` with real form action URLs (e.g., Formspree/Netlify Forms) and your email.
